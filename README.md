# BrewStock

AI inventory system for coffee shop.

## Stack
- Frontend: React + TypeScript
- Backend: FastAPI
- Data: PostgreSQL + Redis
- Forecast: Prophet, XGBoost
- Vision: YOLOv8
- AIaaS: Claude, GPT

## Structure
- `backend/` FastAPI service
- `frontend/` React app scaffold
- `ml/` ML module scaffold
- `infra/` Infra scaffold
